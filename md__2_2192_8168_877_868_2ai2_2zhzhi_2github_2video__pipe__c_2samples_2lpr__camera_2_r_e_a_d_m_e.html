lpr camera

Demonstrate License Plate Recognize(LPR) Camera using VideoPipe.

  1. plate_recognize_pipeline.cpp run a pipeline to detect&recognize license plate and send results via socket (using message broker node in VideoPipe)
  2. plate_recognize_server.py receive plates via socket, then start a web server powered by flask to demonstrate how to display license plate recognize history on UI.
  3. clean.sh help to clean all cache data in current sample directory.
### how to use ###
step1. compile `plate_recognize_pipeline.cpp` and run binary file. (change des_ip first)
step2. run `plate_recognize_server.py` script to receive license plates, then open url in your browser to fetch recognize history. (change `bind_ip` and `live_stream_0` in script first)
### data flow ###
1.[pipeline -> license plate results -> send via socket] -> 2.[receive license plate results -> display results from web]
please run `sh clean.sh` if you want to restart sample from scratch.
